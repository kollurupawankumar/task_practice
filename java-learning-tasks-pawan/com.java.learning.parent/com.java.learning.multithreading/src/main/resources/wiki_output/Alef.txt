The Phoenician letter is derived from an Egyptian hieroglyph depicting an ox's head and gave rise to the Greek Alpha (Î‘), being re-interpreted to express not the glottal consonant but the accompanying vowel, and hence the Latin A and Cyrillic Ğ.
In phonetics, aleph  originally represented the onset of a vowel at the glottis. In Semitic languages, this functions as a weak consonant allowing roots with only two true consonants to be conjugated in the manner of a standard three consonant Semitic root. In most Hebrew dialects as well as Syriac, the glottal onset represented by Aleph is an absence of a true consonant although a glottal stop ([Ê”]), which is a true consonant, typically occurs as an allophone. In Arabic, the Alif has the glottal stop pronunciation when occurring initially.
