A lasso ( or ), from the Castilian word, Lazo. Also called lariat, riata, or reata (all from Castilian, la reata= re-tied rope), is a loop of rope designed as a restraint to be thrown around a target and tightened when pulled. It is a well-known tool of the American cowboy. The word is also a verb; to lasso is to throw the loop of rope around something.
